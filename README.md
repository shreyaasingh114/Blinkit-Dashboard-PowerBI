# Blinkit Sales Dashboard (Power BI)

This is a visually appealing and data-driven Power BI dashboard inspired by Blinkit. It uses Blinkit-branded colors and provides insights into sales, delivery performance, and customer behavior.

## 📊 Key Features

- KPIs like total orders, revenue, and average delivery time
- Category-wise and zone-wise order analysis
- Delivery status breakdown (Delivered, Cancelled, Returned)
- Time-based revenue trends
- Filter by delivery slot

## 🎨 Tech Stack

- Power BI Desktop
- Excel dataset (`BlinkIT Grocery Data.xlsx`)
- Custom theme inspired by Blinkit (Green & Yellow)

## 📁 Files Included

- `BLINKIT DASHBOARD.pbix` – The Power BI dashboard file
- `BlinkIT Grocery Data.xlsx` – The dataset used in the dashboard

## 🧠 Note

The data is fictional and created for academic/portfolio purposes to showcase dashboarding and data visualization skills using Power BI.
